Principais Habilidades

Linguagem - C

C foi a primeira linguagem de programação aprendi, onde a utilizei para dar inicio a um curso de computação. À primeira vista me apaixonei pela linguagem onde podia utilizá-la para fazer o que quisesse utilizando apenas o teminal windows. E foi nessa linguagem que desenvolvi meu primeiro projetinho de programação onde eu e mais 3 amigos desenvolvemos juntos uma cópia do jogo de navegador "Dino", no qual pode ser jogado através do navegador "Google Chrome" caso seja digitado "chrome://dino/" na barra de url.

Linguagem - JavaScript

JavaScript foi a segunda linguagem de programação pelo qual me interessei, visto sua popularidade e quantidade de oportunidades no mercado, no qual estava e ainda está em constante crescente. Até o momento não desenvolvi nenhum projeto seja ele de pequeno a grande porte nessa linguagem, mas posso falar que foi a linguagem que aprendi os conceitos de programação de forma mais rápida e ampla, pode se dizer que foi uma linguagem que me deu novos olhos de como programar de forma mais coesa.

Habilidades com Front-End

Logo após ter estudado muito JavaScript eu percebi de certa forma uma grande dependencia dessa linguagem sobre outras duas, são elas "HTML" e "CSS", percebendo essa dependecia para poder criar páginas web resolvi me aprofundar um pouco nessas linguagens para poder implementar de melhor forma o que desenvolveria em JavaScript. Tenho apenas 1 projeto incompleto sobre essas duas linguagens na qual a partir de curso, eu estava desenvolvendo uma página web responsiva.

Linguagem - Java

Tive contato com java quando cursei uma cadeira sobre linguagem de programação orientada a objetos, e esse foi meu terceiro contato com uma linguagem de programção, onde no projeto final dessa cadeira desenvolvi junto a um parceiro um jogo da memória no estilo clássico onde implementamos vários conceitos de orientação a objetos nele.
